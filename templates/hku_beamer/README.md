# HKU Beamer Template

This folder preserves the source files from the HKU Beamer template used to style the presentation chapter.

## Contents

- `main.tex`: original Beamer template source.
- `figures/hkulogo.png`: HKU logo used by the template and project slides.
- `figures/icecream_vendor.jpg`: sample image included with the template source.

The bilingual presentation slides in `reports/presentation_chapter/zh/` and `reports/presentation_chapter/en/` use this HKU Beamer style while keeping the final project slides separated from the reusable template assets.
