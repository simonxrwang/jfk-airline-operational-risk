\begin{frame}{Where This Chapter Fits}
  % Speaking cue: 30--35 seconds.
  \begin{itemize}
    \item The previous chapter covers sector context, core risk factors, SWOT, and PEST.
    \item This chapter converts BTS raw records into modeling-ready risk assets.
    \item The key question is not only which charts we produced, but which data definitions can support the next models.
  \end{itemize}
  \vspace{0.4em}
  \begin{block}{Modeling handoff}
